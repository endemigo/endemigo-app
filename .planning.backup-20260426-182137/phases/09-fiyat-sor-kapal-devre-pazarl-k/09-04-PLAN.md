# Plan 09-04: Integration — Escrow, Notifications & Order Source

<objective>
Connect negotiation to escrow payment (Phase 6), push notifications (Phase 8), and order tracking (Phase 7). Add source field to Order entity.
</objective>

<must_haves>
- Accepted offer → escrow payment flow
- Order entity source field (DIRECT_SALE | AUCTION | ASK_PRICE)
- Backfill existing auction orders to source: AUCTION
- Push notifications for all 8 negotiation events (D-20)
- Product sold → all conversations auto-close
- 14-day withdrawal right for ASK_PRICE sales
</must_haves>

<tasks>

<task id="1">
<title>Add source field to Order entity</title>
<read_first>
- backend/src/modules/order/entities/order.entity.ts
</read_first>

<acceptance_criteria>
- Order has source column with default DIRECT_SALE
- Migration backfills existing auction orders
</acceptance_criteria>
</task>
