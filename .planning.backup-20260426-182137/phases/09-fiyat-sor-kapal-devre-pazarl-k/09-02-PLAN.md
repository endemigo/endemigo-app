# Plan 09-02: Negotiation Backend Services & Content Moderation

<objective>
Implement NegotiationService (conversation CRUD, offer lifecycle, state machine), ContentModerationService (platform-dışı pattern detection), NegotiationGateway (Socket.IO /negotiation namespace), NegotiationProcessor (BullMQ jobs), and REST controller.
</objective>

<must_haves>
- Conversation create/list/close endpoints
- Offer create/accept/reject/counter endpoints with pessimistic locking
- 6-state machine transitions enforced server-side
- Content moderation with extended pattern set + Unicode normalization
- Socket.IO /negotiation namespace for real-time
- BullMQ jobs for offer expiry and inactivity archival
- Product listing type conflict check (ASK_PRICE vs AUCTION)
- Rate limiting on offer creation
</must_haves>
