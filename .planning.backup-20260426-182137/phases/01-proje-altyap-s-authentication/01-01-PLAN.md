</objective>

<threat_model>
## Security Threat Model (ASVS L1)

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Brute force login | HIGH | Rate limiting: 5 req/min auth endpoints (@nestjs/throttler) |
| Token theft | HIGH | Short-lived access (15min), hashed refresh in DB, rotation |
| SQL injection | HIGH | TypeORM parameterized queries, class-validator on all DTOs |
| XSS via API response | MEDIUM | Helmet security headers, no HTML in API responses |
| Refresh token reuse | HIGH | Reuse detection → revoke all user sessions |
| Credential exposure | HIGH | bcrypt hash (12 rounds), env vars only, no hardcoded secrets |
| Privilege escalation | HIGH | RBAC guard on all protected routes, role check in JWT payload |
| CSRF | LOW | API-only backend (no cookies for auth — Bearer token) |
</threat_model>

```typescript
@Catch()
export class GlobalExceptionFilter implements ExceptionFilter {
  catch(exception: unknown, host: ArgumentsHost) {
    const ctx = host.switchToHttp();
    const response = ctx.getResponse();
    
    let status = 500;
    let message = 'Internal server error';
    let code = 'INTERNAL_ERROR';
    
    if (exception instanceof HttpException) {
      status = exception.getStatus();
      const res = exception.getResponse();
      message = typeof res === 'string' ? res : (res as any).message;
      code = (res as any).code || `HTTP_${status}`;
    }
    
    // Log to Sentry for 5xx
    if (status >= 500) Sentry.captureException(exception);
    
    response.status(status).json({
      success: false,
      error: { code, message, statusCode: status },
      timestamp: new Date().toISOString(),
    });
  }
}
```
